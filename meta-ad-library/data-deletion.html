Data Deletion Instructions

If you believe this project has collected information that should be deleted, submit a request through GitHub repository issues with enough detail for the operator to identify the relevant data.

Please Include

Your request should include:

  • Your name and preferred contact email.
  • A description of the data you want reviewed or deleted.
  • Any relevant URLs, IDs, filing references, or ad identifiers.

Response Process

Requests will be reviewed in a reasonable time frame. Where deletion is required and appropriate, the operator will remove or anonymize the relevant locally stored data, subject to legal, research, or platform-policy obligations.
